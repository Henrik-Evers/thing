Welcome to your Node.js project on Cloud9 IDE!

This chat example showcases how to use `socket.io` with a static `express` server.

## Joining the Server

1) Select the button that say `Preview` (top center), and in the drop-down meny select `Preview Running Application`.

2) Drag the preview tab next to this tab header.

3) In the preview, scroll down and select `Continue to Application`.

## Running the Server

1) Open `server.js` and start the app by clicking on the "Run" button in the top menu.

2) Alternatively you can launch the app from the Terminal:

    $ node server.js

Once the server is running, open the project in the shape of 'https://projectname-username.c9users.io/' (or click `Preview` and then `Preview Running Application`. As you enter your name, watch the Users list (on the left) update. Once you press Enter or Send, the message is shared with all connected clients.
